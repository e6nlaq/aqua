
# 2. Hello, world

## Demonstration

Create a program that outputs `Hello, world` when executed.

## Program

```

# Aqua Learn #2

outf "Hello, world"

```

## Explanation

The function `outf` outputs the contents of a string

So, if you change the following

```

# Aqua Learn #2

outf "Hello, Aqua!"

```

The output will be `Hello, Aqua!`.
